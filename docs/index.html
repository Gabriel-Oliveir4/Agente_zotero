Política de Privacidade — GPT “Bibliotecário AUFTEK”

1. Finalidade

Este GPT auxilia consultas ao acervo de artigos no Zotero do laboratório, retornando respostas com citações/trechos.

2. Dados processados

  • Perguntas enviadas pelos usuários ao GPT.
  • Metadados e trechos retornados pela API (título, ano, DOI/URL, evidências/trechos).

3. Compartilhamento

As perguntas podem ser enviadas à API hospedada no Render e à API do Zotero apenas para recuperar itens e evidências.

4. Armazenamento

Podem existir logs técnicos mínimos no provedor de hospedagem para diagnóstico.

5. Segurança

A API exige autenticação por chave e deve ser usada apenas por membros autorizados do laboratório.
