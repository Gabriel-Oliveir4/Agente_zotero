Política de Privacidade

Esta página descreve como a API “AUFTEK Zotero Proxy” processa dados ao buscar referências e evidências no acervo do Zotero para apoiar validação e citações em respostas.

1) O que este serviço faz

Este serviço fornece endpoints HTTP que consultam a API do Zotero do grupo AUFTEK e retornam (a) referências bibliográficas e (b) trechos (“snippets”) do texto indexado para validação/citação.

2) Dados processados

  • Termos de busca enviados nas requisições (ex.: palavras-chave).
  • Chave do item do Zotero (itemKey) quando solicitada evidência.
  • Metadados e trechos retornados pela API do Zotero.
Recomendação: não envie dados sensíveis (ex.: informações pessoais, dados clínicos identificáveis) como termos de busca.

3) Armazenamento e retenção

O serviço não foi projetado para armazenar conteúdo de usuários nem persistir trechos retornados. Podem existir logs técnicos mínimos do servidor (ex.: data/hora, endpoint e códigos de erro) apenas para diagnóstico e estabilidade.

4) Compartilhamento

O serviço se comunica com a API do Zotero para obter os dados do acervo. Não vendemos dados e não compartilhamos informações com terceiros, exceto o necessário para realizar a consulta ao Zotero.

5) Segurança

As rotas que retornam dados do acervo exigem autenticação via chave de acesso (header X-AGENT-KEY ou Authorization: Bearer), conforme configuração do integrador.

6) Contato

Dúvidas ou solicitações relacionadas a esta política: contato@auftek.com.br
